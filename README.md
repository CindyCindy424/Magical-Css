# Magical-Css
Some interesting special effects made by pure CSS.  ( ¨̮ )  一些用纯CSS实现的有趣特效。
## Inspiration - 想法
 最近看了很多有趣的开源项目，里面有的细节很有趣很有美感，于是得到了一些灵感，做了一些小小的 可以说是特效或者小零件。
 
 在这里记录下来，因为都是很宝贵的idea，而且整理起来也方便日后直接使用。(=^▽^=)
